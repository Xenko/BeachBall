How to Get the BeachBall Add-on for Sandcastle Builder

As a UserScript

  1. If you don't already have it, install the required browser add-on. Tampermonkey (Chrome) or Greasemonkey (Firefox).
  2. Install the script from: http://userscripts.org/scripts/show/183541

As a Bookmarklet

  1. Paste the contents (not the url) of http://xenko.github.io/BeachBall/BeachBall_Bookmark_Loader.js into a bookmark.
  2. Load Sandcastle Builder.
  3. Load the bookmark.

Use the following like to access the BeachBall source code: http://xenko.github.io/BeachBall/BeachBall.js
